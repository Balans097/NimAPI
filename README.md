# NimAPI
Complete reference of Nim standard library functions.

All function descriptions, explanations, and recommendations presented in the references are based on the current source code for Nim standard library modules and do not rely on other sources, including existing references. Therefore, the most accurate and up-to-date information is presented here.


Все представленные в справочниках описания функций, пояснения и рекомендации сформированы на основе текущего исходного кода модулей стандартной библиотеки Nim и не опираются на другие источники, включая существующие справочники. Поэтому здесь представлена наиболее точная и актуальная информация.
